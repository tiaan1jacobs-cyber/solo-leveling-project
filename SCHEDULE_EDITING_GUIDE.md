## **WHAT YOU CAN EDIT**

### **1. Edit Schedule Block (Times & Activity)**

This changes the **schedule structure itself**:
- Start time
- End time
- Activity name
- Description
- Block type (core, training, DA, etc.)
- Whether it's DA job time

**How to Edit a Block:**
1. Click "Manage Schedule"
2. Find the block you want to edit
3. Click the **Clock icon** (üïê)
4. The Schedule Block Editor opens
5. Change any fields you want
6. Click "SAVE BLOCK"

**Example:**
```
Before:

After editing:
6:00 - 6:45 | Morning Ritual & Declarations
```

### **2. Edit Instructions (What to Do)**

This changes the **content inside** the block:
- Declarations to say
- Checklist items
- Step-by-step instructions

**How to Edit Instructions:**
1. Find the block
2. Click the **Edit icon** (‚úèÔ∏è)
3. The Instruction Editor opens
4. Modify declarations, checklist items, etc.
5. Click "SAVE CHANGES"

## **SCHEDULE BLOCK EDITOR FIELDS**

When you click the Clock icon, you can edit:

### **BLOCK ID**
- Unique identifier (e.g., "wake", "meditation", "breakfast")
- **Cannot be changed** after creation (locked field)
- Used to link instructions to this block

### **ACTIVITY NAME**
- What shows on the schedule
- Can be anything you want
- Examples: "Wake Up & Core Declarations", "Combat Training", "Deep Work"

### **START TIME & END TIME**
- 24-hour format (HH:MM)
- Must be valid (end time after start time)
- Examples: 06:30, 14:00, 22:45

### **DESCRIPTION** (Optional)
- Brief explanation shown on schedule
- Examples: "Daily morning routine to start the day strong"

### **BLOCK TYPE**
Choose what kind of activity:
- **Core** = Essential daily activities (green)
- **Training** = Physical training (orange)
- **DA** = DA job work (red)
- **Class** = Academic classes (blue)
- **Business** = Business work (purple)
- **Recovery** = Rest and recovery (indigo)
- **Health** = Health-related (teal)

### **DA JOB FLAG**
- Check this if the block is for DA job work
- Adds a red "DA JOB" badge
- Helps track DA time

### **DISPLAY ORDER**
- Number to control order (0 = first, 1 = second, etc.)
- Lower numbers appear first
- If unsure, leave as 0 and they'll sort by time

1. Click "Manage Schedule"

## **RESET TO DEFAULT SCHEDULE**

If you've customized your schedule but want to go back to the original:

1. Look for **"Reset to Default"** button (top-right)
2. Click it
3. Confirm you want to reset
4. All your custom blocks for that day are deleted
5. Default schedule returns

**Note:** This only affects the current day (e.g., Monday). Other days keep their custom schedules.

## **HOW CUSTOM SCHEDULES WORK**

### **Default Schedule:**
- Built into the app
- Same for all users (unless customized)
- Shows when you haven't created custom blocks

### **Custom Schedule:**
- Saved to YOUR database
- Personal to your account
- Persists forever
- Shows "‚úì Custom Schedule Active" indicator

### **Priority:**
- If you have ANY custom blocks for a day, they replace the entire default schedule
- You can mix: custom blocks on some days, default on others
- Example: Custom schedule Monday-Friday, default on weekends

## **COMMON SCENARIOS**

### **Scenario 1: Change Wake Up Time**

**Goal:** Wake up at 6:00 AM instead of 6:30 AM

1. Click "Manage Schedule"
2. Find "WAKE UP" block
3. Click Clock icon (üïê)
4. Change "Start Time" from 06:30 to 06:00
5. Change "End Time" from 06:35 to 06:05 (if needed)
6. Click "SAVE BLOCK"

**Result:** Wake up block now shows 6:00 - 6:05

### **Scenario 2: Add Evening Study Session**

**Goal:** Add 2-hour study block every evening

1. Click "Manage Schedule"
2. Click "Add New Time Block"
3. Fill in:
   ```
   Block ID: evening_study
   Activity: Evening Study Session
   Start Time: 20:00
   End Time: 22:00
   Type: Academic
   ```
4. Click "SAVE BLOCK"

**Result:** New study block at 8 PM - 10 PM

### **Scenario 3: Mark DA Job Blocks**

**Goal:** Mark all your DA job times

1. Click "Manage Schedule"
2. For each DA work block:
   - Click Clock icon (üïê)
   - Check "This is DA Job Time"
   - Click "SAVE BLOCK"

**Result:** Red "DA JOB" badge on those blocks

### **Scenario 4: Completely Custom Schedule**

**Goal:** Build your own schedule from scratch

1. Click "Manage Schedule"
2. Click "Reset to Default" (start fresh)
3. Click "Add New Time Block" for each block:
   - 6:00 - 6:30: Morning Routine
   - 6:30 - 7:30: Gym
   - 7:30 - 8:00: Breakfast
   - 8:00 - 12:00: Deep Work
   - 12:00 - 13:00: Lunch
   - 13:00 - 17:00: Client Work
   - 17:00 - 18:00: Admin
   - 18:00 - 19:00: Dinner
   - 19:00 - 21:00: Skill Development
   - 21:00 - 22:00: Wind Down
   - 22:00 - 23:00: Reading

4. Add instructions to each block (Edit icon)

**Result:** Completely personalized schedule

## **BEST PRACTICES**

### **1. Start With Defaults**
- Use the default schedule for a few days
- See what works and what doesn't
- Then customize based on experience

### **2. Be Realistic With Times**
- Don't overpack your schedule
- Leave buffer time between blocks
- Account for transitions

### **3. Consistent Block IDs**
- Use descriptive, lowercase names
- No spaces (use underscores)
- Examples: `wake`, `morning_workout`, `deep_work_1`

### **4. Use Block Types Wisely**
- Color-codes help you see schedule at a glance
- Group similar activities with same type
- DA blocks in red = easy to track

### **5. Add Descriptions**
- Brief but helpful
- Explain the "why" not the "what"
- Examples:
  - ‚ùå "Work on business"
  - ‚úÖ "Focus time for client deliverables"

### **6. Order Matters**
- Schedule flows top to bottom by time
- Display order only matters if times are same
- Usually just set to 0

### **7. Test Before Committing**
- Try new schedule for a day
- See if times work
- Adjust as needed
- Don't be afraid to iterate

## **TROUBLESHOOTING**

### **"My changes aren't saving"**
- Make sure you clicked "SAVE BLOCK"
- Check your internet connection
- Look for error toast messages
- Try refreshing the page

### **"I don't see the Manage Schedule button"**
- Make sure you're logged in
- Navigate to Military Command Center view
- Button should be top-right of schedule

### **"I deleted the wrong block"**
- Custom blocks: Can't undo (re-create it)
- Default blocks: Click "Reset to Default" to restore

### **"Times are overlapping"**
- App doesn't prevent overlaps
- Be careful to set non-overlapping times
- Review your schedule after changes

### **"Can't find a block to edit"**
- Use "Manage Schedule" mode
- Clock icon (üïê) = Edit schedule block
- Edit icon (‚úèÔ∏è) = Edit instructions
- Don't confuse the two!
